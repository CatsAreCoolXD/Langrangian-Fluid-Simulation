# Very good fluid simulation
Welcome to the most accurate, most mind-blowing fluid simulation you've ever seen. There is no fluid simulation that is better than mine.
This is a particle-based (Langrangian) fluid simulation made with help from [Sebastian Lague's Fluid Simulation](https://www.youtube.com/watch?v=rSKMYc1CQHE) and his sources.
My goal is to make this a fun sandbox which you can play around with the fluid in.
